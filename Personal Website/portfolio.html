me_ira Bot

This started out as an inside joke between a roommate and myself. This was made mostly to explore some of the web-based applications of python. The bot is able to scrape a subreddit (r/me_ira) for the current top posts. It then checks the posts against a stored list of posts it has previously commented on. If the post does not appear in this list, it will comment with a psuedo-randomly selected entry from a list of possible comments. Click the title to view the GitHub repository.

ODBC Database Manager

Born partly from procrastination and partly from frustration with Microsoft Access. A course I was taking required me to create and manage an Access database, which I was less than thrilled to be doing. So, in comes this program. The GUI interface is driven by the python library tkinter. SQL is scripted to Access with the python library pyodbc. The program features user login with validation, the ability to choose between tables for customer and product data, and the ability to search for, update, add, and delete information within these. Click the title to view the GitHub repository.
